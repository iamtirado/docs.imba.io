# Comments

## Single Line / Inline Comments

```ruby
# This is a single line comment
tag hello-world # It can also go at the end of a line
    def render
        <self> "hello-world"
```

## Multiline Comments / Comment Blocks

```ruby
###
This is a 
multiline comment
###

tag hello-world 
    ###
    Comments can be nested 
    anywhere in your code.
    ### 
    def render
        <self> "hello-world"
```
